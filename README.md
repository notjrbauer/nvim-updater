# nvim-updater

> Install `nightly` nvim with the cli.

## Installation

```
go get github.com/notjrbauer/nvim-updater
```

## Usage

### `--source`

#### Directory to download neovim.

#### Defaults: `cwd`

### `--destination`

#### Directory to symlink neovim.

#### Defaults: `/usr/local/bin`, no need to specify nvim directly.
